难度水平：简单

1. 描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

约束条件：

  • 1 <= strs.length <= 200
  • 0 <= strs[i].length <= 200
  • strs[i] 仅由小写英文字母组成

3. 答案

  • 主要思想：首先使用第一个字符串作为结果，在迭代数组时修改
  • 时间复杂度： O(nm)
  • 空间复杂度： O(m)

m 为最长前缀长度
